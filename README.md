# Permutas API

## Installation

`npm install`
`cd ./ui && npm install`

Create a `.env` file in the root directory, example:

```
SERVER_HOSTNAME=127.0.0.1
SERVER_PORT=3000
DATABASE_URL=mongodb://admin:Admin0.permutas@ds245901.mlab.com:45901/heroku_n3w8jbkr
```

## RUN

- Development: `npm run dev`
- Production: `npm start`
